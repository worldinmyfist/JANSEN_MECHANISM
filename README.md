# JANSEN_MECHANISM
This repo contains code to simulate Jansen mechanism using Newton-Raphson method.

## HOW TO RUN THE CODE
Run this command in MATLAB:
```bash
[tpos,trace] = plot_one_leg(1.57,[2.62,1.4,3.73,2.9,5.26,4.12,5.14,2.81],0.19,200)
```
